# 🏥 CP1 - HandsOn 03

Neste terceiro HandsOn foi proposto a criação de classes para um jogo de cartas genérico, onde futuramente será possível fazer uma a interface para diversos tipos de jogos.

---

# 🛠️ Classes Criadas

## 1. Carta 
A base genérica do projeto, onde existe atributos que contém em qualquer tipo de jogo, que é uma categoria/tipo e um valor/peso.

## 2. Jogador 
É o participante do jogo, que tem uma mão com suas cartas, definidas de forma genérica, de forma que o jogador possa receber cartas de qualquer tipo.

## 3. Baralho
Cria a lógica de qualquer jogo de baralho, embaralhando, adicionando e removendo cartas.

## 4. Rodada
Define a lógica da rodada, recebendo os jogadores, o baralho e a implementa a distribuição de cartas daquela rodada, servindo para qualquer tipo de jogo.
